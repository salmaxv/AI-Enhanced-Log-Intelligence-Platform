# NetOps Copilot 2026

AI-Enhanced Log Intelligence Platform for SWS-212 Server-Side Scripting.

## Project Overview

NetOps Copilot 2026 is a FastAPI-based backend for managing devices, ingesting logs, creating incidents, and generating structured AI-assisted incident reports. The system is designed around an API-first architecture and supports role-based access control for Admin, Operator, and Viewer users.

## Backend Tech Stack

- FastAPI
- Pydantic
- MongoDB
- JWT Authentication
- Passlib (bcrypt)
- Uvicorn
